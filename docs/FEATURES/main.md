# 메인 화면 구성

## 개요

게임 실행 시 최초로 보여지는 메인(타이틀) 화면의 구성을 정의한다. 상세 게임 규칙은 [game_rule.md](./game_rule.md), 미션 1 상세는 [mission1.md](./mission1.md) 참고. 전체 배경은 [../PRD.md](../PRD.md) 참고.

## 화면 구성 요소

- **타이틀 로고**: 화면 상단 중앙에 게임 타이틀("PANG") 표시
- **시작 버튼**: "게임 시작" 버튼 클릭 또는 지정 키 입력 시 Mission 1부터 게임 시작
- **최고 점수(하이스코어) 표시**: 이전 플레이 기록 중 최고 점수를 화면에 노출
- **조작법 안내**: 좌/우 이동, 발사 키 등 기본 조작 안내 문구 표시
- **배경**: 게임의 분위기를 나타내는 정적 배경 이미지 또는 애니메이션(풍선이 떠다니는 연출 등)

## 화면 전환

- 메인 화면 → (시작 버튼/키 입력) → Mission 1 게임 화면으로 전환
- 게임 오버 후 → 최종 점수 표시 화면 → 메인 화면으로 복귀

## 비고

- 본 문서는 초안 수준의 개략적 구성만 다루며, 세부 레이아웃/디자인/애니메이션 스펙은 추후 별도 문서에서 구체화한다.
