# 게임 룰 상세

전체 배경 및 개요는 [../PRD.md](../PRD.md) 참고. 미션별 난이도는 [mission1.md](./mission1.md) 참고.

## 플레이어

- 화면 하단부에서 좌/우로만 이동 가능
- 발사는 위쪽 방향으로만 가능하며, 발사 버튼을 누르면 총알이 플레이어 위치에서 화면 상단을 향해 일정 속도로 날아간다 (즉시 도달이 아닌, 눈에 보이는 이동 모션)
- 풍선과 플레이어가 접촉하면 라이프(목숨)를 1개 소모

## 풍선(버블) 규칙

- 풍선은 화면 내에서 중력의 영향을 받아 포물선 형태로 바운드하며 이동
- 벽이나 지형(블록)에 부딪히면 진행 방향이 반사되어 반대쪽으로 이동
- 바닥(또는 발판)에 닿아 튕길 때마다 이전과 동일한 높이만큼 다시 튀어 오른다 (감쇠 없음)
- 풍선은 대형 → 중형 → 소형 3단계 크기를 가지며, 맞힐 때마다 한 단계 작은 크기의 풍선 2개로 분열한다
- 소형 풍선을 맞히면 분열 없이 즉시 소멸한다
- 스테이지에 배치된 모든 풍선을 소멸시키면 해당 스테이지 클리어

## 지형

- 각 스테이지는 고정된 블록/벽 구조를 가진다
- 일부 지형에는 사다리 등 이동 경로가 포함되어 플레이어의 이동 범위를 제한하거나 확장한다
- 지형 블록은 풍선의 바운드 방향에도 영향을 준다 (벽에 부딪히면 반사)

## 라이프(목숨) 시스템

- 게임 시작 시 라이프 5~10개 보유(초기값은 추후 확정)
- 라이프를 모두 소진하면 게임 오버
- 일정 점수 달성 시 추가 라이프 획득

## 점수 및 스테이지 진행

- 풍선을 소멸시키면 점수 획득
- 한 스테이지의 모든 풍선을 제거하면 다음 스테이지로 자동 진행
- 총 50개 스테이지로 구성되며, 스테이지가 진행될수록 풍선 개수/지형 복잡도가 증가

## 게임 종료 조건

- 라이프를 모두 소진 → 게임 오버 → 최종 점수 표시
- 전체 스테이지(50개)를 모두 클리어 → 엔딩
