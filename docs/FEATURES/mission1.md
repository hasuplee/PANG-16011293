# Mission 1

전체 게임 룰은 [game_rule.md](./game_rule.md) 참고. 전체 배경은 [../PRD.md](../PRD.md) 참고.

## 개요

Mission 1은 게임의 첫 번째 스테이지로, 플레이어가 게임 시작 후 가장 먼저 진입하는 스테이지다. 이후 스테이지들의 기준점이 되는 가장 낮은 난이도로 설계한다.

## 난이도 설정

- **풍선 개수**: 대형 풍선 1~2개만 배치하여 기본 조작(이동/발사)과 분열 규칙(대형 → 중형 → 소형)을 학습시키는 단계
- **풍선 크기/속도**: 가장 크고 느린 풍선으로 구성하여 바운드 궤적을 예측하기 쉽게 함
- **지형**: 별도의 벽/블록 없이 단순한 평면 바닥으로 구성 (사다리 등 이동 제약 요소 없음)
- **제한 시간**: 없음 (별도 타이머 없이 풍선을 모두 제거할 때까지 진행)

## 규칙 적용

- [game_rule.md](./game_rule.md)에 정의된 기본 규칙(중력에 의한 바운드, 벽 반사, 접촉 시 라이프 소모, 대형 → 중형 → 소형 분열 및 소멸 방식)을 동일하게 적용
- 스테이지 내 모든 풍선을 제거하면 Mission 2로 자동 진행

## 목적

- 신규 플레이어가 조작감(좌우 이동, 발사 타이밍)과 풍선 바운드 패턴을 자연스럽게 익히도록 하는 튜토리얼 성격의 스테이지
