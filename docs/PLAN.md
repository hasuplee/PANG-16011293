# PLAN - Phase별 개발 계획

이 문서는 팡(Pang) 게임을 Phase 단위로 나누어 개발하기 위한 계획이다. **각 Phase는 그 자체로 실제 브라우저에서 동작하는 SW**를 목표로 하며, Phase 완료 시 고객님이 직접 `npm run dev`로 실행해보고 해당 Phase에서 추가된 기능을 테스트/피드백하는 것을 전제로 한다.

관련 문서: [PRD.md](./PRD.md), [FEATURES/main.md](./FEATURES/main.md), [FEATURES/game_rule.md](./FEATURES/game_rule.md), [FEATURES/mission1.md](./FEATURES/mission1.md)

---

## Phase 1. 메인 화면

- **목표 기능**: 게임 실행 시 메인 화면이 표시된다. 타이틀 로고와 함께 게임 선택 / 시작 버튼 / 게임 종료 메뉴가 노출된다. 시작 버튼을 누르면 게임 화면으로 진입한다(이 시점에는 빈 화면 또는 준비 중 화면으로 대체 가능). 게임 종료 버튼을 누르면 종료 동작이 수행된다.
- **고객님 테스트 포인트**:
  - 실행 시 메인 화면이 바로 뜨는지
  - 게임 선택 / 시작 / 종료 메뉴가 모두 화면에 보이고 클릭 가능한지
  - 시작 버튼 클릭 시 화면이 전환되는지
  - 종료 버튼 클릭 시 의도한 동작(종료 처리)이 일어나는지

## Phase 2. 플레이어 이동

- **목표 기능**: 게임 화면에 플레이어 캐릭터가 표시되고, 좌/우 키 입력으로 이동한다. 화면 경계 밖으로는 나가지 않는다.
- **고객님 테스트 포인트**:
  - 캐릭터가 좌/우로 부드럽게 움직이는지
  - 화면 끝에서 멈추고 더 이상 나가지 않는지
  - 이동 속도가 게임 조작감상 너무 느리거나 빠르지 않은지

## Phase 3. 발사 및 풍선 이동

- **목표 기능**: 발사 키를 누르면 캐릭터 위치에서 위쪽으로 무기(와이어)가 발사되어 화면 상단까지 도달한다. 대형 풍선 1개가 등장하여 중력의 영향을 받아 바운드하며 움직이고, 벽에 부딪히면 반사된다. (아직 충돌 판정은 없음 - 무기와 풍선이 서로 그냥 통과)
- **고객님 테스트 포인트**:
  - 발사 시 와이어가 즉시 화면 상단까지 나가는지
  - 풍선이 바닥에 닿을 때마다 이전과 같은 높이로 튀어 오르는지 (자연스러운 바운드감)
  - 풍선이 벽에 부딪혔을 때 반대 방향으로 잘 튕기는지

## Phase 4. 풍선 분열 규칙

- **목표 기능**: 무기가 풍선에 맞으면 판정이 동작한다. 대형 풍선을 맞히면 중형 2개로 분열, 중형을 맞히면 소형 2개로 분열, 소형을 맞히면 분열 없이 소멸한다.
- **고객님 테스트 포인트**:
  - 대형 → 중형 → 소형 순서로 정확히 분열되는지
  - 분열된 풍선 2개가 서로 반대 방향으로 튀어나가는지
  - 소형 풍선을 맞혔을 때 완전히 사라지는지

## Phase 5. 플레이어 충돌 및 라이프 시스템

- **목표 기능**: 풍선에 플레이어가 닿으면 라이프(목숨)가 1개 감소한다. 라이프를 모두 소진하면 게임 오버 화면이 표시된다. 화면에 남은 라이프 개수가 UI로 표시된다.
- **고객님 테스트 포인트**:
  - 풍선과 캐릭터가 닿는 판정이 체감상 정확한지 (너무 예민하거나 둔감하지 않은지)
  - 라이프 UI 숫자가 줄어드는 게 잘 보이는지
  - 라이프가 0이 되었을 때 게임 오버 화면으로 잘 전환되는지

## Phase 6. 점수 및 Mission 1 클리어 (전체 흐름 완성)

- **목표 기능**: 풍선을 소멸시키면 점수가 올라가고 화면에 표시된다. Mission 1에 배치된 모든 풍선을 제거하면 스테이지 클리어 화면이 표시된다. 메인 화면(Phase 1) → 게임 시작 → Mission 1 플레이 → 게임 오버/클리어 → 메인 화면 복귀까지 전체 흐름이 연결된다.
- **고객님 테스트 포인트**:
  - 점수가 풍선을 맞힐 때마다 정확히 올라가는지
  - 모든 풍선을 없앴을 때 클리어 화면이 뜨는지
  - 메인 화면부터 게임 오버/클리어 후 메인 화면 복귀까지 전체 흐름이 끊기지 않는지
  - Mission 1을 처음부터 끝까지 플레이했을 때 전체적으로 게임답게 느껴지는지 (난이도, 속도감 등 종합 피드백)

Mission 1 이후의 Mission 2 이상 스테이지, 추가 아이템/무기, 하이스코어 저장 등은 Phase 6 완료 후 고객님 피드백을 반영하여 별도 Phase로 추가한다.
